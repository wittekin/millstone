# Security Policy

## Supported Versions

Security fixes are applied to the latest stable release line.

| Version | Supported |
| ------- | --------- |
| Latest release line | Yes |
| Older release lines | No |

## Reporting a Vulnerability

Please do not open public issues for undisclosed vulnerabilities.

Preferred channel:
1. GitHub private vulnerability reporting: https://github.com/wittekin/millstone/security/advisories/new

If private advisories are unavailable, contact maintainers privately via GitHub profile contact details.

Please include:
- Affected version/commit
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if known)

## Response Process

- We will acknowledge receipt as quickly as possible.
- We will triage, reproduce, and prioritize the report.
- We will coordinate disclosure and remediation updates.
